# MVP 范围

## 目标

构建 Repo Scout 的第一个可用版本，帮助 AI 初学者发现值得学习、关注和尝试的 GitHub 开源项目。

## 核心用户流程

1. 打开首页。
2. 浏览 GitHub 热门项目榜单。
3. 通过关键词搜索项目。
4. 进入项目详情页。
5. 查看项目为什么适合初学者或值得关注。

## MVP 功能

- 首页展示热门项目榜单。
- 支持按项目名、描述、主题和语言搜索。
- 提供 AI 精选项目专题栏。
- 项目详情页展示核心元数据。
- 后端 API 支持榜单、搜索、精选专题和项目详情。
- 使用数据库存储项目数据。
- 提供手动执行的 GitHub Trending 数据抓取命令。

## 暂不纳入 MVP

- 用户账号。
- 收藏或个人项目集。
- 消息通知。
- 浏览器插件。
- GitHub Trending 以外的多数据源抓取。
- 完整推荐系统。

## 成功标准

- 用户能从首屏理解这个产品是做什么的。
- 用户能通过榜单和搜索找到项目。
- 热门数据存储在数据库中，而不是硬编码在页面里。
- 精选项目和原始 Trending 榜单分开存储。
- 抓取流程可以重复执行，并且不会重复创建同一个项目。
